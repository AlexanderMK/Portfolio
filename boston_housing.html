Data description analysis¶
1. They are : ZN (proportion of residential land zoned for lots over 25,000 sq.ft.) with 0 for 25th, 50th percentiles.¶
2. CHAS: Charles River dummy variable (1 if tract bounds river; 0 otherwise) with 0 for 25th, 50th and 75th percentiles.¶
These coulms may not be useful in regression task such as predicting MEDV (Median value of owner-occupied homes)¶
The max value of MEDV seems to be censored at 50.00 (corresponding to a median price of $50,000). Values above 50.00 may not help to predict MEDV.¶
